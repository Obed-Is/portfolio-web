# Portafolio de Obed Aquino

Portafolio web personal desarrollado con Vue.js 3 y Vite, proyectos y habilidades como desarrollador junior.

## Stack Tecnológico

- Vue.js 3
- Vite
- JavaScript
- CSS3

## Instalación

```bash
git clone https://github.com/tu-usuario/portfolio.git
cd portfolio
pnpm install
pnpm dev
```
